# Todo

※ 本ファイルはあくまでタスクリスト。仕様としてのタスクではない。
テストの各タスク名はテスト名に転記する。実際の仕様はテストコードに移譲する。
全てのタスクが完了したら、本ファイルはアーカイブ予定。

## テストリスト

```markdown
- [ ]：未着手
- [|]：進行中（途中で終わる場合）
- [x]：完了
- [>]：別のセクションにタスク起票済み
```

- [ ] タスク(単体)
  - [>] ユーザーの負担にならないよう、入力か所のほとんどはnull許容として作成できる。
    - [>] タイトルは 0 文字を許容
    - [>] 期限なしを許容
    - [>] 開始日時なしを許容
    - [>] 完了日時なしを許容
    - [>] キャンセル日時なしを許容
  - [>] 進行状態を先行して変更することを許容する（あくまでデータとしての更新日は実際の変更日とは異なるため）
    - [>] 開始日時 < 更新日時 を許容
    - [>] 完了日時 < 更新日時 を許容
    - [>] キャンセル日時 < 更新日時 を許容

  - [x] タスク作成
    - [x] 種類別のタスク作成
      - [x] 未着手タスク作成
      - [x] 進行中タスク作成
      - [x] 完了タスク作成
      - [x] キャンセルタスク作成
    - [x] タイトル指定
      - [x] 0文字タイトルを許容
      - [x] 255文字タイトルを許容
      - [x] 256文字タイトルを拒否
    - [x] 期限の指定
      - [x] 期限なしを許容
      - [x] 期限ありを許容
    - [x] 開始日時の指定
      - [x] 開始日時なしを許容
      - [x] 開始日時ありを許容
    - [x] 完了日時の指定
      - [x] 完了日時なしを許容
      - [x] 完了日時ありを許容
    - [x] キャンセル日時の指定
      - [x] キャンセル日時なしを許容
      - [x] キャンセル日時ありを許容
    - [x] 作成日時
      - [x] 作成日時は外から注入
    - [x] 更新日時
      - [x] 更新日時は未指定可能(作成日時 == 更新日時)
      - [x] 更新日時は外部から注入可能

    - [x] 時間関係
      - [-] 期限に制約なし
        - [-] 期限と開始日時に制約無し
        - [-] 期限と完了日時に制約無し
        - [-] 期限とキャンセル日時に制約無し
        - [-] 期限と作成日時に制約無し
        - [-] 期限と更新日時に制約無し
      - [x] 開始日時との関係
        - [x] 開始日時と完了日時
          - [x] 開始日時 < 完了日時を許容
          - [x] **開始日時 == 完了日時を許容**
          - [x] **開始日時 > 完了日時を拒否**
        - [-] 開始日時とキャンセル日時との間に制約なし
        - [-] 開始日時と作成日時との間に制約なし
        - [-] 開始日時と更新日時との間に制約なし
      - [-] 完了日時との関係
        - [-] 完了日時とキャンセル日時との間に制約なし
        - [-] 完了日時と作成日時との間に制約なし
        - [-] 完了日時と更新日時との間に制約なし
      - [-] キャンセル日時との関係
        - [-] キャンセル日時と作成日時との間に制約なし
        - [-] キャンセル日時と更新日時との間に制約なし
      - [x] 作成日時との関係
        - [x] 作成日時と更新日時
          - [x] 作成日時 <= 更新日時を許容
          - [x] 作成日時 > 更新日時を拒否

  - [ ] 編集
    - [x] タイトルの編集
      - [x] タスクはタイトルを変更できる
      - [x] 0文字のタイトルを許容
      - [x] 255文字のタイトルを許容
      - [x] 256文字のタイトルを拒否
    - [x] 期限の編集
      - [x] 期限の変更を許容
      - [x] 期限なしへの変更を許容
      - [x] 期限ありへの変更を許容
    - [x] 開始日時の編集を許容
    - [ ] 完了日時の編集を許容
    - [ ] キャンセル日の編集
    - [ ] 状態の編集
      - [ ] 未着手を、開始時刻を与えて進行中状態に変更する。
      - [ ] 未着手を、開始時刻と完了日時を与えて完了状態に変更する。
      - [ ] 未着手を、キャンセル日時を与えてキャンセル状態に変更する。
      - [ ] 進行中を未着手に戻せる。その際、開始日時は削除される。
      - [ ] 進行中を、完了日時を与えて完了状態に変更する。
      - [ ] 進行中を、キャンセル日時を与えてキャンセル状態に変更する。
      - [ ] 完了を、開始日時・完了日時の情報を落として未着手に戻せる。
      - [ ] 完了を、完了日時の情報を落として進行中に戻せる。
      - [ ] 完了を、開始日時を与えて進行中状態に変更する。
      - [ ] 完了を、開始日時とキャンセル日時を与えてキャンセル状態に変更する。

- [ ]

- Given-When-Then の形になっていない（特に When と Then）
