# 与件

- 超絶シンプルなタスクアプリ

## 作りたいもの

- タスク
  - 共通
    - タイトル: 文字列 (100文字以内)
    - 期限: 日時 | なし
    - 作成日時: 日時
    - 更新日時: 日時 (> 作成日時)
    - 状態: 未着手 | 進行中 | 完了 | キャンセル
    - 開始日?: 日時 | なし
    - 完了日?: 日時 | なし
    - キャンセル日?: 日時

  - 未着手のタスク
    - 状態: 未着手
    - 更新日時 = 作成日時
    - × 開始日
    - × 完了日
    - × キャンセル日

  - 進行中のタスク
    - 状態: 進行中
    - 開始日: 日時
    - × 完了日
    - × キャンセル日

  - 完了したタスク
    - 状態: 完了
    - 開始日: 日時
    - 完了日: 日時
    - × キャンセル日

  - キャンセルされたタスク
    - 状態: キャンセル
    - 開始日: 日時 | なし
    - 完了日: 日時 | なし
    - キャンセル日: 日時

- タスクの状態遷移
    <------------------->
  未着手 <-> 進行中 <-> 完了
    ↕         ↕        ↕
          キャンセル
  状態遷移で日付などのなくなるプロパティの情報は保たない。

  - 未着手 -> 進行中
    - 開始日: 現在日時
  - 進行中 -> 完了
    - 完了日: 現在日時
  - 進行中 -> キャンセル
    - キャンセル日: 現在日時
  - 未着手 -> キャンセル
    - キャンセル日: 現在日時
  - キャンセル -> 未着手
    - 開始日: なし
    - 完了日: なし
    - キャンセル日: なし

- タスクのCRUD
  - タスクの作成
  - タスクの更新
    - タイトルの変更
    - 期限の変更
    - 状態の変更
  - タスクの削除


## 意識すること
