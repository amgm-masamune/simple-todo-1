# 与件

- 超絶シンプルなタスクアプリ

## 作りたいもの

- タスク
  - 共通
    - タイトル
    - 期限: 日時 | なし
    - 作成日時: 日時
    - 更新日時: 日時

  - 未着手のタスク
    共通 ×
    - 状態: 未着手

  - 着手済みのタスク
    共通 ×
    - 状態: 着手中
    - 開始日: 日時

  - 完了したタスク
    共通 ×
    - 状態: 完了
    - 開始日: 日時
    - 完了日: 日時

  - キャンセルされたタスク
    共通 ×
    - 状態: キャンセル
    - 開始日: 日時 | なし
    - 完了日: 日時 | なし
    - キャンセル日: 日時

- タスクの状態遷移
  - 未着手 → 着手中
  - 着手中 → 完了
  - 未着手 → 完了
  - 未着手 → キャンセル
  - 着手中 → キャンセル
  - 完了 → キャンセル
  - キャンセル → 未着手
  - キャンセル → 着手中
  - キャンセル → 完了

## 守るもの

- テスト可能な設計にする
